Problem D6.1 ‚Äî Mass‚ÄìSpring SHM (Undamped)#

A mass \(m = 0.50~\mathrm{kg}\) is attached to a spring with stiffness

\[ k = 200.0~\mathrm{N/m}. \]

The mass is released from rest at a displacement

\[ x_0 = 0.10~\mathrm{m}, \]

so the initial velocity is zero.

The angular frequency for a mass‚Äìspring system undergoing simple harmonic motion is given by

\[ \omega = \sqrt{\frac{k}{m}}. \]

Answer the following:

  1. What is the angular frequency \(\omega\)?

  2. What is the period \(T\) of the motion?

  3. What is the frequency \(f\)?

Be careful with units and significant figures.

Problem solution

Relevant equations

Angular frequency for a mass‚Äìspring oscillator:

\[ \omega = \sqrt{\frac{k}{m}} \]

Relationships between angular frequency, period, and frequency:

\[ \omega = 2\pi f, \qquad T = \frac{1}{f}. \]

Part 1 ‚Äî Angular frequency

Given

  • \(k = 200.0~\mathrm{N/m}\)

  • \(m = 0.50~\mathrm{kg}\)

Compute \(\omega\)

\[ \omega = \sqrt{\frac{200.0}{0.50}} = \sqrt{400} = 20.0~\mathrm{rad/s}. \]

With proper significant figures:

\[ \boxed{\omega = 20.0~\mathrm{rad/s}} \]

Part 2 ‚Äî Period

Use the relationship between period and angular frequency:

\[ T = \frac{2\pi}{\omega}. \]

Substitute the value of \(\omega\):

\[ T = \frac{2\pi}{20.0} = 0.314~\mathrm{s}. \]

With proper significant figures:

\[ \boxed{T = 0.314~\mathrm{s}} \]

Part 3 ‚Äî Frequency

The frequency is the inverse of the period:

\[ f = \frac{1}{T}. \]

Substitute the value of \(T\):

\[ f = \frac{1}{0.314} = 3.18~\mathrm{Hz}. \]

Equivalently, using \(\omega = 2\pi f\):

\[ f = \frac{\omega}{2\pi} = \frac{20.0}{2\pi} = 3.18~\mathrm{Hz}. \]

With proper significant figures:

\[ \boxed{f = 3.18~\mathrm{Hz}} \]

Final Summary

  • The angular frequency depends only on the system parameters \(k\) and \(m\), not on amplitude.

  • The period tells how long one complete oscillation takes.

  • The frequency tells how many oscillations occur each second.

These quantities describe the timing of the oscillation, independent of how far the mass is displaced.

Problem D6.2 ‚Äî Small-Angle Pendulum#

A simple pendulum of length

\[ L = 1.00~\mathrm{m} \]

oscillates with small amplitude, so the small-angle approximation applies.
The angular frequency of a simple pendulum in this regime is given by

\[ \omega = \sqrt{\frac{g}{L}}. \]

Answer the following:

  1. What is the angular frequency \(\omega\)?

  2. What is the period \(T\) of the motion?

Use \(g = 9.81~\mathrm{m/s^2}\).
Be careful with units and significant figures.

Problem solution

Relevant equations

Angular frequency for a small-angle pendulum:

\[ \omega = \sqrt{\frac{g}{L}} \]

Relationship between angular frequency and period:

\[ T = \frac{2\pi}{\omega} \]

Part 1 ‚Äî Angular frequency

Given

  • \(L = 1.00~\mathrm{m}\)

  • \(g = 9.81~\mathrm{m/s^2}\)

Compute \(\omega\)

\[ \omega = \sqrt{\frac{9.81}{1.00}} = \sqrt{9.81} = 3.13~\mathrm{rad/s}. \]

With proper significant figures:

\[ \boxed{\omega = 3.13~\mathrm{rad/s}} \]

Part 2 ‚Äî Period

Use the relationship between period and angular frequency:

\[ T = \frac{2\pi}{\omega}. \]

Substitute the value of \(\omega\):

\[ T = \frac{2\pi}{3.13} = 2.01~\mathrm{s}. \]

With proper significant figures:

\[ \boxed{T = 2.01~\mathrm{s}} \]

Final Summary

  • The angular frequency of a small-angle pendulum depends only on its length and the local value of gravity.

  • The period is independent of the mass and (for small angles) the amplitude.

  • These results describe the timing of the oscillation, not the forces that cause it.

Problem D6.3 ‚Äî Traveling Wave Parameters#

A periodic disturbance travels along a string with

\[ \lambda = 0.80~\mathrm{m}, \qquad f = 5.0~\mathrm{Hz}. \]

Answer the following:

  1. Find the wavenumber \(k\).

  2. Find the period \(T\).

  3. Find the phase speed \(v_p\).

Be careful with units and significant figures.

Problem solution

Relevant equations

Wavenumber:

\[ k = \frac{2\pi}{\lambda} \]

Period‚Äìfrequency relationship:

\[ T = \frac{1}{f} \]

Phase speed:

\[ v_p = f\lambda \]

Part 1 ‚Äî Wavenumber

Given

  • \(\lambda = 0.80~\mathrm{m}\)

Compute \(k\)

\[ k = \frac{2\pi}{0.80} = 7.85~\mathrm{rad/m}. \]

With proper significant figures:

\[ \boxed{k = 7.9~\mathrm{rad/m}} \]

Part 2 ‚Äî Period

Given

  • \(f = 5.0~\mathrm{Hz}\)

Compute \(T\)

\[ T = \frac{1}{5.0} = 0.20~\mathrm{s}. \]

With proper significant figures:

\[ \boxed{T = 0.20~\mathrm{s}} \]

Part 3 ‚Äî Phase speed

Compute \(v_p\)

\[ v_p = (5.0)(0.80) = 4.0~\mathrm{m/s}. \]

With proper significant figures:

\[ \boxed{v_p = 4.0~\mathrm{m/s}} \]

Final Summary

  • The wavenumber describes how rapidly the wave pattern repeats in space.

  • The period describes how often the oscillation repeats in time.

  • The phase speed tells how fast the wave pattern moves along the string.

These quantities together fully characterize the basic timing and spatial structure of a traveling wave.

Problem D6.4 ‚Äî Traveling Wave from Spatial and Temporal Repetition#

A wave repeats in space with wavenumber

\[ k = 12~\mathrm{rad/m}, \]

and repeats in time with period

\[ T = 0.40~\mathrm{s}. \]

Answer the following:

  1. Find the wavelength \(\lambda\).

  2. Find the frequency \(f\).

  3. Find the phase speed \(v_p\).

Be careful with units and significant figures.

Problem solution

Relevant equations

Wavelength‚Äìwavenumber relationship:

\[ \lambda = \frac{2\pi}{k} \]

Frequency‚Äìperiod relationship:

\[ f = \frac{1}{T} \]

Phase speed:

\[ v_p = f\lambda \]

Part 1 ‚Äî Wavelength

Given

  • \(k = 12~\mathrm{rad/m}\)

Compute \(\lambda\)

\[ \lambda = \frac{2\pi}{12} = 0.524~\mathrm{m}. \]

With appropriate significant figures:

\[ \boxed{\lambda = 0.52~\mathrm{m}} \]

Part 2 ‚Äî Frequency

Given

  • \(T = 0.40~\mathrm{s}\)

Compute \(f\)

\[ f = \frac{1}{0.40} = 2.5~\mathrm{Hz}. \]

With appropriate significant figures:

\[ \boxed{f = 2.5~\mathrm{Hz}} \]

Part 3 ‚Äî Phase speed

Compute \(v_p\)

\[ v_p = (2.5)(0.52) = 1.3~\mathrm{m/s}. \]

With appropriate significant figures:

\[ \boxed{v_p = 1.3~\mathrm{m/s}} \]

Final Summary

  • The wavenumber sets the spatial repetition of the wave.

  • The period (or frequency) sets the temporal repetition of the oscillation.

  • The phase speed connects these spatial and temporal descriptions, giving the speed of the traveling wave pattern.

These quantities describe the structure and timing of the wave without reference to the forces that produce it.

Problem D6.5 ‚Äî Read Frequency from an Oscillation Graph#

The figure below shows the displacement \(x(t)\) of a single oscillator versus time.

Task:

  1. From the graph, estimate the period \(T\) by measuring the time between two successive peaks
    (use several cycles and average).

  2. Compute the frequency using

    \[ f = \frac{1}{T}. \]

Report your answers with appropriate units.

Problem solution

Key idea

For oscillatory motion, the period is the time required for one full cycle of motion. On a displacement‚Äìtime graph, this can be measured as the time between successive peaks, troughs, or any two identical points in the cycle.

Using multiple cycles and averaging improves accuracy.

Part 1 ‚Äî Period

From the graph, successive peaks occur at approximately

\[ t = 0.40~\text{s},\ 0.80~\text{s},\ 1.20~\text{s}, \ldots \]

The time between peaks is therefore about

\[ T \approx 0.40~\text{s}. \]

Averaging over several cycles gives a consistent value of

\[ \boxed{T \approx 0.40~\text{s}}. \]

Part 2 ‚Äî Frequency

The frequency is the reciprocal of the period:

\[ f = \frac{1}{T}. \]

Substituting the measured value of \(T\),

\[ f = \frac{1}{0.40} = 2.5~\text{Hz}. \]

With appropriate significant figures:

\[ \boxed{f \approx 2.5~\text{Hz}}. \]

Final Summary

  • The period is read directly from the time separation between repeating features on the graph.

  • The frequency tells how many oscillations occur each second.

  • Graphical measurements are approximate, so results should reflect reasonable precision based on the scale of the graph.

import numpy as np
import matplotlib.pyplot as plt

A = 0.8             # amplitude (units of x)
f_true = 1.50       # Hz (do not reveal to students; they should read from the graph)
phi = 0.35          # phase offset (rad)

# Time base
t = np.linspace(0, 3.2, 1200)  # a bit over 3 seconds for multiple cycles
x = A * np.cos(2*np.pi*f_true*t + phi)

# Plot (clean, grid for reading off peaks)
plt.figure(figsize=(8,4))
plt.plot(t, x, linewidth=2)
plt.xlabel("Time $t$ (s)")
plt.ylabel("Displacement $x(t)$ (arb. units)")
plt.title("Oscillation $x(t)$ ‚Äî Estimate Period $T$ from Peak-to-Peak")
plt.grid(True, which="both", alpha=0.4)
plt.xlim(t.min(), t.max())
plt.show()

%reset -f
import numpy as np
from IPython.display import Markdown, display, Math

A = 0.8             # amplitude (units of x)
f_true = 1.50       # Hz (do not reveal to students; they should read from the graph)
phi = 0.35          # phase offset (rad)

# Time base
t = np.linspace(0, 3.2, 1200)  # a bit over 3 seconds for multiple cycles
x = A * np.cos(2*np.pi*f_true*t + phi)

# --- Peak detection: simple local-maximum finder ---
# A point i is a peak if x[i-1] < x[i] > x[i+1]
idx = np.where((x[1:-1] > x[:-2]) & (x[1:-1] > x[2:]))[0] + 1
t_peaks = t[idx]

# Use interior peaks to avoid edge effects, then compute average spacing
if len(t_peaks) >= 3:
    T_estimates = np.diff(t_peaks)         # successive peak spacings
    T_est = T_estimates.mean()
    f_est = 1.0 / T_est
else:
    T_est = np.nan
    f_est = np.nan

# Nicely formatted output
display(Markdown(r"""Read the period *T* as the time between **successive peaks** on the plot (average over several cycles).  
Then compute the frequency using
"""))
display(Math(r"$$f=\frac{1}{T}.$$"))

print(f"Estimated period from graph (avg peak spacing): T ‚âà {T_est:.1e} s")
print(f"Estimated frequency: f ‚âà {f_est:.1e} Hz")

Problem D6.6 ‚Äî Read \(\lambda\) and \(T\) from Traveling-Wave Graphs#

Two figures are provided:

  1. A spatial snapshot \(y(x,t_0)\) at a fixed time \(t_0\) (displacement vs. position).

  2. A time trace \(y(x_0,t)\) at a fixed position \(x_0\) (displacement vs. time).

Tasks:

  1. From the spatial snapshot, estimate the wavelength \(\lambda\) by measuring the horizontal distance between successive crests (average across several).

  2. From the time trace, estimate the period \(T\) by measuring the time between successive crests (average across several).

  3. Compute the frequency:

\[ f=\frac{1}{T}. \]

  1. Compute the phase speed:

\[ v_p=\frac{\lambda}{T}. \]

Report your answers with units. Show your reading strategy (which peaks you chose) and your averaging.

Problem solution

Key ideas

  • In a spatial snapshot \(y(x,t_0)\), time is fixed and the pattern repeats in space. The horizontal spacing between repeating features (such as crests) is the wavelength \(\lambda\).

  • In a time trace \(y(x_0,t)\), position is fixed and the motion repeats in time. The time spacing between repeating features is the period \(T\).

  • Once \(\lambda\) and \(T\) are measured, the frequency and phase speed follow from

\[ f=\frac{1}{T},\qquad v_p=\frac{\lambda}{T}. \]

Part 1 ‚Äî Wavelength from the spatial snapshot

Choose several crests on the \(y(x,t_0)\) graph. For example, suppose crests occur at approximately

\[ x = 0.40~\text{m},\ 1.20~\text{m},\ 2.00~\text{m},\ 2.80~\text{m}. \]

Compute successive crest-to-crest spacings:

\[ \Delta x_1 = 1.20-0.40 = 0.80~\text{m} \]
\[ \Delta x_2 = 2.00-1.20 = 0.80~\text{m} \]
\[ \Delta x_3 = 2.80-2.00 = 0.80~\text{m} \]

Average these values to estimate \(\lambda\):

\[ \lambda \approx \frac{\Delta x_1+\Delta x_2+\Delta x_3}{3} = 0.80~\text{m}. \]

So,

\[ \boxed{\lambda \approx 0.80~\text{m}}. \]

Part 2 ‚Äî Period from the time trace

Now choose several crests on the \(y(x_0,t)\) graph. For example, suppose crests occur at approximately

\[ t = 0.30~\text{s},\ 0.70~\text{s},\ 1.10~\text{s},\ 1.50~\text{s}. \]

Compute successive time spacings:

\[ \Delta t_1 = 0.70-0.30 = 0.40~\text{s} \]
\[ \Delta t_2 = 1.10-0.70 = 0.40~\text{s} \]
\[ \Delta t_3 = 1.50-1.10 = 0.40~\text{s} \]

Average these values to estimate \(T\):

\[ T \approx \frac{\Delta t_1+\Delta t_2+\Delta t_3}{3} = 0.40~\text{s}. \]

So,

\[ \boxed{T \approx 0.40~\text{s}}. \]

Part 3 ‚Äî Frequency

Compute

\[ f=\frac{1}{T}=\frac{1}{0.40}=2.5~\text{Hz}. \]

So,

\[ \boxed{f \approx 2.5~\text{Hz}}. \]

Part 4 ‚Äî Phase speed

Compute

\[ v_p=\frac{\lambda}{T}=\frac{0.80}{0.40}=2.0~\text{m/s}. \]

So,

\[ \boxed{v_p \approx 2.0~\text{m/s}}. \]

Final Summary

  • The wavelength is read from the spatial repetition in \(y(x,t_0)\).

  • The period is read from the temporal repetition in \(y(x_0,t)\).

  • Frequency and phase speed follow directly from \(f=1/T\) and \(v_p=\lambda/T\).

Because the measurements are read from a graph, results should reflect reasonable precision based on the axis scales.

import numpy as np
import matplotlib.pyplot as plt

A = 1.0            # amplitude (arb. units)
lam_true = 0.75    # wavelength (m)
f_true = 2.0       # frequency (Hz)
T_true = 1.0 / f_true
vp_true = lam_true * f_true


# Spatial snapshot y(x, t0)
t0 = 0.10
x = np.linspace(0, 3*lam_true, 1200)
y_space = A * np.cos(2*np.pi*(x/lam_true) - 2*np.pi*f_true*t0)

plt.figure(figsize=(8,4))
plt.plot(x, y_space, linewidth=2)
plt.xlabel("Position $x$ (m)")
plt.ylabel("$y(x,t_0)$ (arb. units)")
plt.title("Spatial Snapshot $y(x,t_0)$ ‚Äî Estimate Wavelength $\\lambda$")
plt.grid(True, alpha=0.4)
plt.xlim(x.min(), x.max())
plt.show()

# Time trace y(x0, t)

def find_peak_positions(xvals, yvals):
    """Return x-positions of simple local maxima (exclude endpoints)."""
    idx = np.where((yvals[1:-1] > yvals[:-2]) & (yvals[1:-1] > yvals[2:]))[0] + 1
    return xvals[idx]

# 1) Wavelength from spatial snapshot
x_peaks = find_peak_positions(x, y_space)
# Use interior peaks and average spacing
lam_estimates = np.diff(x_peaks)
lam_est = np.mean(lam_estimates) if len(lam_estimates) >= 1 else np.nan

# 2) Period from time trace

T_estimates = np.diff(t_peaks)
T_est = np.mean(T_estimates) if len(T_estimates) >= 1 else np.nan

# 3) Frequency and 4) Phase speed
f_est = 1.0 / T_est if np.isfinite(T_est) and T_est > 0 else np.nan
vp_est = lam_est / T_est if np.isfinite(lam_est) and np.isfinite(T_est) and T_est > 0 else np.nan

Problem D6.7 üêçüíª ‚Äî Plotting a Traveling Wave: Snapshot vs. Time Trace#

A traveling wave along a string is described by

\[ y(x,t) = 0.20\cos(2\pi x - 4\pi t), \]

where \(x\) is measured in meters and \(t\) is measured in seconds.

You will use Python to generate two different plots of this same wave and interpret what each plot represents.

Tasks:

  1. Create a spatial snapshot \(y(x,t_0)\) by holding time fixed at

    \[ t_0 = 0. \]

    Plot displacement versus position for

    \[ 0 \le x \le 2.0~\text{m}. \]

  2. Create a time trace \(y(x_0,t)\) by holding position fixed at

    \[ x_0 = 0.25~\text{m}. \]

    Plot displacement versus time for

    \[ 0 \le t \le 1.0~\text{s}. \]

  3. From your spatial snapshot, estimate the wavelength \(\lambda\).

  4. From your time trace, estimate the period \(T\).

  5. Use your measured values to compute the phase speed

\[ v_p=\frac{\lambda}{T}. \]

Label your axes clearly and include units.

Problem solution

Key idea

The same wave can appear very different depending on what is held fixed:

  • Holding time fixed shows the wave‚Äôs spatial structure.

  • Holding position fixed shows the wave‚Äôs oscillatory motion in time.

Part 1 ‚Äî Spatial snapshot

Set \(t_0=0\). The wave becomes

\[ y(x,0)=0.20\cos(2\pi x). \]

Plot this expression versus \(x\).
From the plot, successive crests are separated by approximately

\[ \lambda \approx 1.0~\text{m}. \]

Part 2 ‚Äî Time trace

Set \(x_0=0.25~\text{m}\). The wave becomes

\[ y(0.25,t)=0.20\cos\!\big(2\pi(0.25)-4\pi t\big). \]

Plot this expression versus \(t\).
From the plot, successive crests occur every

\[ T \approx 0.50~\text{s}. \]

Part 3 ‚Äî Phase speed

Using the measured values,

\[ v_p=\frac{\lambda}{T} =\frac{1.0}{0.50} =2.0~\text{m/s}. \]

So,

\[ \boxed{v_p \approx 2.0~\text{m/s}}. \]

Final Summary

  • A snapshot reveals spatial repetition and wavelength.

  • A time trace reveals temporal repetition and period.

  • The phase speed connects these two perspectives and describes how fast the wave pattern moves.

This problem emphasizes that graphs of waves show patterns, not particles moving through space.
